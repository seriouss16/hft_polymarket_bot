Polymarket HFT Arbitrage & Trend Bot
Данный проект представляет собой высокочастотного торгового бота, эксплуатирующего задержку (latency) между обновлением цен на централизованных биржах (Binance, Coinbase, Kraken) и исполнением ордеров на децентрализованной платформе Polymarket.

🧩 Основные принципы работы
Latency Arbitrage (Арбитраж задержки)
Бот использует WebSocket-фиды с прямым доступом к книгам ордеров (L1/L2) ведущих бирж. Цена на Polymarket (основанная на оракулах Chainlink) обновляется с задержкой от 1 до 5 секунд по сравнению со спотовым рынком. Бот входит в позицию, "предсказывая" движение оракула на основе уже случившегося движения на Binance.

Гибридный анализ (Strategy Layers)

Tactical Layer (ms): Сравнение текущего взвешенного среднего (Weighted Mid) цен Binance/Coinbase с лучшим Ask/Bid в стакане Polymarket.

Strategic Layer (sec): LSTM-модель, работающая в фоновом режиме, анализирует тренд на отрезке 100 последних тиков для фильтрации входов против глобального движения.

Smoothing Layer: Индикатор RSI предотвращает вход в сделку на экстремальных значениях (перекупленность/перепроданность), минимизируя риск "разворота об бота".

Dynamic Slot Switching
Бот автоматически вычисляет текущий 5-минутный рынок BTC (например, btc-updown-5m-2200) и переключает WebSocket-подписки и token_id без остановки системы.

⚡️ Технологический стек
Язык: Python 3.10+ (asyncio).

ML: TensorFlow/Keras (LSTM Regressor).

Data: WebSockets (JSON/Binary parsing).

Execution: Polymarket CLOB API.
